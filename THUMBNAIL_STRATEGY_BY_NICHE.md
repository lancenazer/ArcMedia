## 📺 VISIONARY BUILDS (Mega Infrastructure / Construction)

### Niche Focus
- Engineering marvels, skyscrapers, infrastructure projects
- Geographic: Saudi Arabia, China, USA, emerging markets

### Thumbnail Formula

| Element | Implementation | Example |
|---------|---------------|---------|
| **Main Visual** | Buildings, cranes, construction sites | Jeddah Tower, China city |
| **Text Focus** | Numbers (cost, height) + "BILLION" | "$500 BILLION", "1KM TALL" |
| **Colors** | Gold/blue (construction) + Red (urgency) | Skyline + gold text |
| **Emotion** | Awe, shock, disbelief | "BACK FROM THE DEAD" |
| **Hooks** | Numbers, comparison, "record" | "RECORD SPEED" |

### Nano Banana Template
```
Create a YouTube thumbnail: [construction/mega project], [dramatic sky/lighting], bold [COLOR] text [HOOK NUMBER/WORD] in industrial font, [secondary visual], high contrast, cinematic, 16:9
```

### Do's ✅
- Use buildings/structures as main visual
- Always include a number ($, height, year)
- Gold/blue for epic construction feel
- Add question mark for curiosity

### Don'ts ❌
- No faces (not relevant to niche)
- Don't use peaceful imagery — needs drama
- Avoid outdated architecture styles

## ⚓ ANCHOR WATCH (Naval / Military / Defense)

### Niche Focus
- Warships, aircraft carriers, weapons systems
- US vs China comparisons

### Thumbnail Formula

| Element | Implementation | Example |
|---------|---------------|---------|
| **Main Visual** | Ships, weapons, flags | Carrier at sea, jets |
| **Text Focus** | Cost + Rankings + "VS" | "$13 BILLION", "TOP 5", "VS" |
| **Colors** | Navy blue + Red (military) + Gold (expensive) | Ship + gold text |
| **Emotion** | Power, dominance, threat | "GAME CHANGER" |
| **Hooks** | Comparison, rankings, fear | "STRONGEST NAVY?" |

### Nano Banana Template
```
Create a YouTube thumbnail: [military ship/weapon], [ocean/dramatic backdrop], bold [COLOR] text [HOOK] in industrial stencil font, [flag/nationality element], high contrast, military photography style, 16:9
```

### Do's ✅
- Ships at sea with dramatic waves
- Country flags (USA, China for comparison)
- Numbers: "$13 BILLION", rankings
- VS graphics for comparisons

### Don'ts ❌
- Don't show peaceful naval imagery
- Avoid boring ship photos
- No outdated military equipment

## 🎬 FILM LORE (Celebrity / Entertainment / Gossip)

### Niche Focus
- Celebrity scandals, downfalls, industry secrets
- Trending news + pattern analysis

### Thumbnail Formula

| Element | Implementation | Example |
|---------|---------------|---------|
| **Main Visual** | Celebrity faces (public figures) | Shocked expression |
| **Text Focus** | "HID", "TRUTH", "SHOCKING", "EXPOSED" | "THEY HID THIS" |
| **Colors** | Red/Yellow (scandal) + Black | Red spotlight, black bg |
| **Emotion** | SHOCK, scandal, revelation | Mouth open, eyes wide |
| **Hooks** | Exposé, hidden truth, downfall | "What They HID" |

### Nano Banana Template
```
Create a YouTube thumbnail: [celebrity face], [shocked/surprised expression], dramatic [COLOR] lighting, bold text [HOOK] in industrial stencil font, [emotion element], high contrast, studio photography, 16:9
```

### Do's ✅
- Celebrity faces with MAX emotion (shocked, angry, sad)
- Red colors = scandal/danger
- Question marks for curiosity
- "THEY HID THIS" proven evergreen hook

### Don'ts ❌
- Don't use neutral expressions
- Avoid calm, peaceful imagery
- No smiling faces (unless mocking)
- Don't use for private individuals

## 🔑 UNIVERSAL RULES (All Channels)

From THUMBNAIL_GUIDE.md:

### Must Have
- [ ] **Curiosity gap** — Creates burning question
- [ ] **High contrast** — Colors pop at small sizes
- [ ] **3-5 words max** — Text readable mobile
- [ ] **Faceless alternatives** — Objects work when faces don't

### AIDA Application
| Letter | Application |
|--------|-------------|
| **A** | Thumbnail grabs attention (colors, faces) |
| **I** | Creates interest (curiosity gap) |
| **D** | Drives desire to click (question must be answered) |
| **C** | Call to action (must click to find out) |

### For Faceless (Visionary, Anchor)
- Use objects: buildings, ships, weapons
- Numbers: costs, rankings, measurements
- Flags: country identity
- Question marks: curiosity
