# WEEKLY CONTENT WORKFLOW
## ArcMedia Group - Using Our Guides Every Week

**Process:** Every week, before creating content, complete these steps in order.

---

## PHASE 1: IDEA GENERATION (Before Anything Else)
**Guide:** `VIRAL_IDEA_GENERATION.md`

### Step 1: Competitor Scan (30 min)
- [ ] Check top 3 channels in our niche
- [ ] Note last 6 months outlier videos (3-50x views)
- [ ] Document what made them work

### Step 2: Content Gap Research (20 min)  
- [ ] Read comments on top videos
- [ ] Use AnswerThePublic for questions
- [ ] Note gaps we can fill

### Step 3: Bold Concept Brainstorm (20 min)
- [ ] Generate 10 ideas using viral formulas
- [ ] Select top 3 based on criteria

---

## PHASE 2: TITLE & CONCEPT REFINEMENT
**Guide:** Both guides work together here

### Step 4: Title Creation (per video)
Using VIRAL_IDEA_GENERATION:
- [ ] Create 4 title variations
- [ ] Apply curiosity gap + specific numbers
- [ ] Use proven viral structure

Using THUMBNAIL_GUIDE:
- [ ] Verify title + thumbnail create curiosity gap
- [ ] Ensure thumbnail creates burning question

### Step 5: Thumbnail Brief
- [ ] Create designer brief
- [ ] Add AI prompts as backup
- [ ] Run through verification checklist

---

## PHASE 3: EXECUTION

### Step 6: Research File
- [ ] Create research file in `weeks/WEEK_X/research/`
- [ ] Include: facts, talking points, sources, hooks

### Step 7: Script Creation
- [ ] Write script using hook + AIDA
- [ ] Verify against title promise

### Step 8: Thumbnail Creation
- [ ] Send brief to designer
- [ ] OR generate with AI prompts
- [ ] Verify against checklist

---
